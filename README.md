# 电子发票金额提取器

这是一个简单的Web应用程序，用于从电子发票PDF文件名中提取金额并计算总和。

## 功能特点

- 支持拖拽上传PDF文件
- 最多支持同时处理100个文件
- 自动从文件名中提取金额
- 显示详细的计算过程
- 计算总金额

## 安装说明

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 打开浏览器访问：`http://localhost:5000`

## 使用方法

1. 将PDF文件拖拽到指定区域，或点击选择文件
2. 系统会自动从文件名中提取金额
3. 查看计算过程和总金额

## 支持的金额格式

- 普通数字：123.45
- 带￥符号：￥123.45
- 带"元"字：123.45元
